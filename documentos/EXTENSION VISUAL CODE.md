# Extensión de Visual Studio Code

## Nombre de la extensión

**Simulador de carga DB**

Esta extensión de Visual Studio Code forma parte del proyecto **Simulador de Carga de Base de Datos**. Su objetivo es permitir ejecutar la simulación de carga de datos directamente desde el entorno de desarrollo, facilitando el uso del proyecto mediante una interfaz integrada en VS Code.

---

## Conclusión

La extensión **Simulador de carga DB** permite integrar el funcionamiento del simulador directamente dentro de Visual Studio Code. Esto facilita la ejecución del proyecto, mejora la experiencia del usuario y permite realizar la simulación de carga de datos de una manera más rápida y práctica.
